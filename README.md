## What is this?
The first independent, dual-framework governance and alignment 
benchmark of five frontier AI models.

## Key Finding
All five models scored NON-COMPLIANT on TRUE-10 (25–28/100) 
while scoring strongly on ALIGN100 (0.84+).
This is the Governance-Alignment Gap.
